# Neural BI - Frontend

This is the frontend application for **Neural BI**, an AI-powered conversational business intelligence dashboard. 

## 🚀 Technologies
- **React (Vite)**
- **Tailwind CSS** (for styling)
- **Lucide React** (for iconography)
- **Framer Motion** (for animations)
- **Axios** (for API communication)

## ✨ Features
- **Neural Glassmorphism UI**: A premium, modern interface with sleek gradients and blur effects.
- **Conversational Analytics**: Interact with your data using natural language.
- **Dynamic Projections**: Instant generation of charts (Pie, Bar, Area) based on AI synthesis.
- **Encrypted Session Persistence**: Your selected dataset and chat history persist across refreshes using `localStorage`.
- **Intelligent Auth Redirection**: Secure handling of user sessions with automatic redirection from login/signup for authenticated users.

## 🛠️ Setup Instructions
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the dashboard at `http://localhost:3000` (or the port specified by Vite).
